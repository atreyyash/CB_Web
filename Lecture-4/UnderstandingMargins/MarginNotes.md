# Margin 
```
// This will acquire same margin around the element
margin: 30px;
```

```
// Margin can also be defined seperately in the following way :-
margin-top: 10px;
margin-bottom: 40px;
margin-right: 10px;
margin-left: 90px;
```
```
// T = Top, B = Bottom
// L = Left, R = Right
        T/B  L/R
margin: 30px 60px;
```

```
         T    R    B    L
margin: 30px 80px 40px 10px;
```
